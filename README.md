
En esta actividad aprendí a utilizar etiquetas básicas de HTML para crear páginas web sencillas.
También aprendí a utilizar formularios, imágenes, botones y hojas de estilo CSS para mejorar el diseño visual de una página.
Además, comprendí cómo conectar diferentes archivos HTML mediante enlaces y cómo organizar correctamente un proyecto web dentro de Visual Studio Code.
Gracias a esta práctica fue posible comprender mejor el funcionamiento básico del desarrollo web y la importancia del diseño visual en una página.
